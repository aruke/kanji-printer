### ðŸ’» Build Locally

1.  Clone the project.
2.  `cd` into `generator` directory, and run `node main.js`.
3.  Wait for it to run. It may take upto 15 minutes based on the computer's processing power.
4.  `cd` to project root & run `npm run start` to run the web app.
